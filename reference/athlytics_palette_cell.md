# Cell Journal Palette

Modern palette based on Cell Press visualization standards. Balances
professional appearance with visual clarity.

## Usage

``` r
athlytics_palette_cell()
```

## Value

A character vector of 8 hex color codes

## Examples

``` r
# Get Cell palette colors
colors <- athlytics_palette_cell()
colors[1] # Blue
#> [1] "#0173B2"
```
